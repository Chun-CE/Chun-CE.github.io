Markdown学习笔记

发表于 2020-01-28 更新于 2020-02-14 分类于 杂学笔记
创建博客后的第一篇博客,主要记录了学习Markdown的一些笔记.

Markdown编辑器

工欲善其事,必先利其器.在进行Markdown的学习之前,通过朋友推荐,我选择了Visual Studio Code 作为Markdown编辑器.
vs code官网下载
下载步骤很常规,不再赘述

Markdown常用语法

标题

# 一级标题
## 二级标题
### 三级标题

…

显示效果

一级标题

二级标题

三级标题

列表

无序列表

或-或+加空格加内容 注意- + *与内容之间的 **空格*

例

+ 我是第一行
- 我是第二行
* 我是第三行
* 我是第四行

效果

  • 我是第一行
  • 我是第二行
  • 我是第三行
  • 我是第四行

有序列表

数字加.加空格
1.加空格加内容
2.加空格加内容
…

注意 . 与文字之间的 空格

  1. 我是第一行
  2. 我是第二行
  3. 我是第三行
  4. 我是第四行

列表嵌套

在上一级和下一级之间敲一个tab 或 四个空格即可

例

  1. 有序列表1
    • 无序列表1
      • 无序列表1.1
        • 无序列表1.1.1
          • 无序列表1.1.1.1
            • 无序列表1.1.1.1.1
      • 无序列表1.2
    • 无序列表2
      • 无序列表2.1
      • 无序列表2.2
  2. 有序列表2
    • 无序列表1
      • 无序列表1.1
      • 无序列表1.2
    • 无序列表2
      • 无序列表2.1
      • 无序列表2.2
  3. 有序列表3
    1. 有序列表3.1
      1. 有序列表3.1.1
      2. 有序列表3.1.2
      3. 有序列表3.1.3
    2. 有序列表3.2
      1. 有序列表3.2.1
        1. 有序列表3.2.1.1
          • 无序列表3.2.1.1.1

字体

加粗

分别用两个将要加粗的文字包裹 **与文字之间不要有 **空格*

**这是要加粗的文字**
这是加粗后的文字

斜体

分别用一个将要倾斜的文字包裹 *与文字之间不要有 **空格*

*这是要倾斜的文字*
这是倾斜后的文字

加粗倾斜

分别用三个*将要加粗倾斜的文字包裹
\**与文字之间不要有 **空格
\这是要加粗倾斜的文字***
这是加粗倾斜后的文字

删除线

分别用两个~~号将要删除的文字包裹
例
~~这是要加删除线的文字~~

效果
这是加了删除线的文字

分割线

三个或三个以上的*或-

* 可紧跟文字下一行

- 需空一行(如果不空将会成为标题)

例:

快要分割了(使用)
**
显示效果:
快要分割了

例:
又快要分割了(使用-)
-—
显示效果
又快要分割了

Markdown特殊语法

代码

单行代码

代码之间分别用一个反引号(tab上面的那个)包起来
可在``外加*进行 加粗 倾斜

例
**`System.out.println(“Hello World!!”)`**
效果
System.out.println("Hello World!!")

多行代码

代码之间分别用三个反引号包起来,且两边的反引号单独占一行

例
```
while(true)
{
System.out.println(Hello World!!);
}
```
效果 

while(true)  
{  
    System.out.println(Hello World!!);   
}

区块(引用)

在引用的文字前加 > 即可，引用可以嵌套，如加两个>>三个>>>

例：
>引用的内容
>>引用的内容2
>>>引用的内容3

效果

引用的内容1

引用的内容2

引用的内容3

普通链接

方法:

或者

\https://Chun-CE.github.io\

使用变量代替链接,文末附带变量地址(很类似于C语言中的声明函数)
例

这个链接用 百度 作为网址变量 [百度][百度]
…

[百度]: http://www.baidu.com/

效果

这个链接用 百度 作为网址变量 百度
…

…

图片

语法格式:
![alt 属性文本](图片地址)
![alt 属性文本](图片地址 “可选标题”)

  • 开头一个感叹号!
  • 接着一个方括号,里面放上图片的代替文字
  • 接着一个普通括号,里面放上图片的网址,最后还可以用引号包住并加上选择性的’title’属性的文字
    例
    ![RUNOOB 图标](http://static.runoob.com/images/runoob-logo.png “RUNOOB”)
    效果

和链接类似,图片也可以使用变量表示

这个图片用 1 作为网址变量 [RUNOOB][1]
然后在文档的结尾为变量赋值(网址)

[1]: http://static.runoob.com/images/runoob-logo.png

这个图片用 2 作为网址变量 [RUNOOB][2]

[2]: http://static.runoob.com/images/runoob-logo.png

如果要显示本地图片,只需将网址换为图片路径.

对齐方式

-\: 内容和标题栏右对齐
\:- 内容和标题栏左对齐
\:-\: 内容和标题栏居中

例

| 左对齐 | 居中 | 右对齐|
| :—- | :—-: | —-: |
| 单元格单元格单元格 | 单元格单元格单元格 | 单元格单元格单元格 |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

效果
