世人都说vim好.

什么是vim?

Vim是从 vi 发展出来的一个文本编辑器。代码补完、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用。
简单的来说， vi 是老式的字处理器，不过功能已经很齐全了，但是还是有可以进步的地方。 vim 则可以说是程序开发者的一项很好用的工具。
连 vim 的官方网站 (http://www.vim.org) 自己也说 vim 是一个程序开发工具而不是文字处理软件。

vim的使用

motion

motion: 表示操作范围的指令

  • num + motion = num个motion
按键 含义
h 左
j 下
k 上
l 右
0 光标定位至行首
^ (脱字符)同上
$ 光标定位至行尾
b 光标定位至光标所在单词的起始处
e 光标定位至光标所在单词的结尾处
w 光标定位至下一个单词的起始处
gg 光标定位至文件开头
G 光标定位至文件末尾

模式

基本模式 派生模式
普通模式 操作符等待模式
插入模式 插入普通模式
可视模式 插入可视模式
选择模式 插入选择模式
命令行模式 替换模式
Ex模式 \

插入模式:

敲击按键 含义
i 在光标左插入
I 在光标所在行的行首插入
a 在光标右插入
A 在光标所在行的行尾插入
o 在光标所在行的下方插入空行
O 在光标所在行的上方插入空行
s 删除光标指定的字符并插入
S 删除光标所在行并插入

常用命令

按键 命令 技巧
d 删除 d + num + motion = 删除num个motion
vim中的删除其实是将内容移到寄存器中,相当于剪切功能
y 拷贝 y + num + motion = 拷贝num个motion
p 粘贴 小写的p是粘贴至光标后,大写的P是光标前
r 替换 替换当前单个字符,R进入替换模式
c 修改 删除当前选中并进入插入模式
u 撤销 撤销最后一次修改,U撤销当前行修改
% 定位括号 快速找到另一半括号
ctrl+r 恢复 恢复撤销的内容
ctrl+g 文件信息 查看文件信息
行号+G 跳转 跳转值行号处
>> 缩进 num+>> = 缩进num行
<< 反缩进 num+<< = 反缩进num行
:! shell :!+命令,如,:! ls
:w 文件名 另存为 按 v 进入可视模式,选中后:w 文件名为局部另存为,加!为强制执行
:r 文件名 合并文件 其实是把另一个文件内容复制到你的光标后
-o 打开多个文件 vim -o test1 test2,-o垂直,-o水平
ctrl + w + w切换至下一个文件,ctrl + w + 方向切换文件,:wqa保存并退出
查找 / ?
搜索方向 从光标向后 从光标向前
n 向后搜索下一个 向前搜索下一个
N 向前搜索下一个 向后搜索下一个
. * [ ] ^ % / ? ~ $ 前需加反斜杠 \
替换 :s :s/old/new 将光标所在行第一个old替换为new
:s/old/new/g 将光标所在行的所有old替换为new
:%s/old/new/g 将整个文件中所有的old替换为new
:%s/old/new/gc 同上,但在每次替换前先请示
:5,13s/old/new/g 替换第5行到第13行中所有的old为new

拷贝/删除与motion

  • d + num + motion = 删除num个motion
  • y + num + motion = 拷贝num个motion
按键 按键 命令
x 删除单字符
y0 d0 (拷贝/删除)当前位置到该行行首
y^ d^ 同上
y$ d$ (拷贝/删除)当前位置到该行行尾
yb db (拷贝/删除)当前位置到单词起始处
ye de (拷贝/删除)当前位置到单词结尾处
yw dw (拷贝/删除)当前位置到下个单词起始处
yh dh (拷贝/删除)左边第一个字符
yl dl (拷贝/删除)右边第一个字符
yj dj (拷贝/删除)所在行及下一行
yk dk (拷贝/删除)所在行及上一行
yy dd (拷贝/删除)当前行
ygg dgg (拷贝/删除)当前行到文件开头
yG dG (拷贝/删除)当前行到文件结尾
